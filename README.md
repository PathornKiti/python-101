# Python Learning Path

This repository contains a structured Python learning path for students who want to learn Python from beginner level to practical application development.

The learning path is divided into two main courses:

1. **Beginner Course: Python Fundamentals**
2. **Intermediate Course: Practical Python Programming**

The beginner course focuses on basic Python syntax and problem-solving.

The intermediate course focuses on building real Python projects using scripts, modules, command line tools, testing, logging, APIs, visualization, and simple applications.

Detailed unit outlines, exercises, project notebooks, and capstone instructions are stored inside each course folder. The standalone intermediate course outline is available in [`python_for_intermediate_class/`](python_for_intermediate_class/).

---

## Course Overview

| Course | Level | Main Focus | Final Outcome |
|---|---|---|---|
| Beginner Course: Python Fundamentals | Beginner | Python syntax, logic, data structures, functions, NumPy, and OOP basics | Students can solve beginner Python problems and build small notebook-based projects |
| Intermediate Course: Practical Python Programming | Intermediate | Project structure, `.py` files, command line, Git, virtual environments, testing, logging, APIs, visualization, reports, and Streamlit | Students can build structured Python applications and small portfolio projects |



## Overall Timeline

Each week is estimated at approximately **2 hours**.

| Course | Duration | Estimated Hours | Main Output |
|---|---:|---:|---|
| Beginner Course: Python Fundamentals | 20 weeks | 40 hours | Notebook exercises, Project 1, Project 2, and Capstone 1 |
| Intermediate Course: Practical Python Programming | 14 weeks | 28 hours | Structured CLI project, data analysis bridge, and practical Python capstone |
| Total Learning Path | 34 weeks | 68 hours | Beginner-to-intermediate Python portfolio |

# Course 1: Beginner Course — Python Fundamentals

## Course Summary

The beginner course is designed for students who are new to Python.

Students learn Python step by step through beginner-friendly explanations, daily-life examples, practice exercises, reference solutions, and cheat sheets.

The course starts from basic data types and ends with a beginner-level capstone project.

---

## Beginner Course Timeline

Each week is estimated at approximately **2 hours**.

| Week | Unit / Project | Topic | Main Focus |
|---:|---|---|---|
| Week 1 | Unit 01 | Python Data Types | Variables, numbers, strings, booleans, arithmetic, type conversion |
| Week 2 | Unit 02 | Selection | `if`, `elif`, `else`, comparison operators, Boolean logic |
| Week 3 | Unit 03 Part 1 | Repetition: `for` Loops | Counting, summing, repeated actions |
| Week 4 | Unit 03 Part 2 | Repetition: `while` Loops | Sentinel loops, stopping conditions, loop control |
| Week 5 | Project 1 | Foundation Project | Apply Unit 01–03 together |
| Week 6 | Unit 04 | Strings | Text processing, slicing, cleaning, formatting |
| Week 7 | Unit 05 | Files | Reading files, writing files, processing records |
| Week 8 | Unit 06 Part 1 | Lists | List creation, indexing, updating, looping |
| Week 9 | Unit 06 Part 2 | Lists and Review | Sorting, summarizing, mixed list practice |
| Week 10 | Unit 07 Part 1 | Tuples and Sets | Fixed records, unique values, set operations |
| Week 11 | Unit 07 Part 2 | Dictionaries | Key-value lookup, counting, grouping data |
| Week 12 | Project 2 | Data Container Project | Apply Unit 04–07 together |
| Week 13 | Unit 08 Part 1 | Functions | Parameters, return values, reusable logic |
| Week 14 | Unit 08 Part 2 | Functions and Recursion | Function design, scope, basic recursion |
| Week 15 | Unit 09 Part 1 | NumPy Basics | Arrays, vectorized calculations, indexing |
| Week 16 | Unit 09 Part 2 | NumPy Data Analysis | Boolean masks, aggregation, 2D arrays |
| Week 17 | Unit 10 Part 1 | Classes and Objects | Class, object, attribute, method, `self`, `__init__` |
| Week 18 | Unit 10 Part 2 | OOP Concepts | Encapsulation, abstraction, inheritance, polymorphism |

| Week 20 | Capstone Review | Presentation and Feedback | Improve and present final project |

# Course 2: Intermediate Course — Practical Python Programming

## Course Summary

The intermediate course is designed for students who already understand Python fundamentals.

The goal is to help students move from notebook-based coding to structured Python development.

Students learn how to organize Python projects, create `.py` files, build reusable libraries, use the command line, manage environments, write tests, log program behavior, work with JSON/CSV, call APIs, build reports, and create simple Streamlit applications.

---

## Intermediate Course Timeline

Each week is estimated at approximately **2 hours**.

| Week | Unit / Project | Topic | Main Focus |
|---:|---|---|---|
| Week 1 | Unit 01 Part 1 | From Notebook to `.py` Files | Convert notebook code into Python scripts |
| Week 2 | Unit 02 | Create Your Own Library | Build reusable modules and import your own function, class, and data model |
| Week 3 | Unit 03 Part 1 | Command Line Basics | Run Python scripts from terminal and use basic shell commands |
| Week 4 | Unit 03 Part 2 | Git and Virtual Environment | Use Git, `.gitignore`, `venv`, `pip`, and `requirements.txt` |
| Week 5 | Unit 04 Part 1 | Error Handling and Debugging | Use `try/except`, read errors, and debug code |
| Week 6 | Unit 04 Part 2 | Testing, Logging, JSON, and CSV | Add tests, logs, and saved data files |
| Week 7 | Project 1 | Structured CLI Project | Build a structured command-line application |
| Week 8 | Unit 05 | Data Preparation and Analysis Bridge | Clean Project 1 data and prepare summaries for later reports, charts, Streamlit, APIs, and capstone planning |
| Week 9 | Unit 06 Part 1 | Visualization and Reports | Create charts and summary reports |
| Week 10 | Unit 06 Part 2 | Streamlit Basics | Turn a report into a simple Streamlit app |
| Week 11 | Unit 07 Part 1 | API Basics | Call an API and parse JSON responses |
| Week 12 | Unit 07 Part 2 | API App with Error Handling | Build a small API-powered app |
| Week 13 | Capstone Planning | Intermediate Capstone Design | Plan project structure, features, and data flow |
| Week 14 | Capstone Build | Intermediate Capstone Project | Complete and present a practical Python application |

## Expected Outcome

After completing both courses, students should be able to:

- write Python programs from scratch
- solve beginner and intermediate programming problems
- organize code into notebooks, scripts, modules, and projects
- use Git and virtual environments
- handle errors and debug programs
- test important logic
- save and load data using JSON and CSV
- call external APIs
- create simple charts, reports, and Streamlit apps
- prepare for specialized paths such as data analysis, automation, app development, or AI
