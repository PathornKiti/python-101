# Python Learning Path

This repository contains a structured Python learning path from beginner fundamentals to intermediate programming.

The learning path is divided into two main courses:

1. **Beginner Course: Python Fundamentals**
2. **Intermediate Course: Intermediate Python Programming**

The beginner course focuses on Python syntax, logic, data structures, functions, NumPy, and OOP basics.

The intermediate course focuses on building real Python programs using project structure, error handling, debugging, JSON/CSV, OOP, testing, and command-line applications.

---

# Course 1: Python Fundamentals

## Course Overview

The beginner course is designed for students who are new to Python.

Students will learn basic Python syntax and gradually build the ability to solve daily-life problems using code.

Each unit includes:

- beginner-friendly explanations
- daily-life examples
- practice exercises
- reference solutions
- cheat sheet tables
- milestone projects

# Course 2: Intermediate Python Programming

## Course Overview

The intermediate course is designed for students who already understand Python fundamentals.

The goal is to help students move from:

> “I know Python syntax”

to:

> “I can build structured, maintainable Python programs.”

This course is inspired by the idea of a **Day 21–60 project-based Python path**.

The focus is general programming, not only data analysis.

Students will learn:

- project structure
- better function design
- modules and imports
- error handling
- debugging
- JSON and CSV persistence
- intermediate OOP
- testing
- command-line applications
- complete application projects

# Recommended Learning Outcome

After completing both the beginner and intermediate courses, students should be able to:

- write Python programs from scratch
- organize code into functions, modules, and projects
- handle user input safely
- debug common errors
- read and write JSON and CSV files
- design simple class-based applications
- build command-line programs
- test important logic
- prepare for specialized paths such as data analysis, app development, automation, or AI
